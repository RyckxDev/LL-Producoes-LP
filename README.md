# 🎬 LL Produções — Landing Page

> Landing page da **LL Produções**, produtora audiovisual com foco em conteúdo criativo e de alto impacto.

---

## 📌 Sobre o Projeto

Este repositório contém a **landing page oficial da LL Produções**, desenvolvida com o objetivo de apresentar os serviços, portfólio e identidade visual da empresa para potenciais clientes e parceiros.

A página foi construída de forma simples, leve e responsiva, sem dependência de frameworks externos.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** — Estrutura semântica da página
- **CSS3** — Estilização, responsividade e animações
- **JavaScript (Vanilla)** — Interatividade e dinamismo

---

## 📁 Estrutura do Projeto

```
LL-Producoes-LP/
├── index.html        # Página principal
├── css/
│   └── style.css     # Estilos globais
├── js/
│   └── main.js       # Scripts de interação
└── assets/
    └── images/       # Imagens e mídias
```

> A estrutura pode variar ligeiramente conforme a organização do repositório.

---

## ✨ Funcionalidades

- Apresentação da LL Produções
- Seção de serviços oferecidos
- Portfólio de trabalhos realizados
- Informações de contato
- Layout responsivo para dispositivos móveis e desktop

---

## 🚀 Como Visualizar

1. Clone o repositório:
   ```bash
   git clone https://github.com/RyckxDev/LL-Producoes-LP.git
   ```
2. Abra o arquivo `index.html` diretamente no seu navegador.

Não é necessário instalar dependências ou configurar ambiente.
